Quick tour

A few things to know before you start.

  1. Pick a poem from the “Select a poem…” dropdown in any panel.
  2. Switch panel type (Transcription / IIIF Viewer / Companion) from the panel header.
  3. Compare witnesses using the witness selector (Transcription) or witness buttons (Viewer).
  4. Arrange the workspace: drag the ☰ handle to reorder, use × to collapse, and drag gutters to resize.
  5. Toggle theme with the sun/moon icon in the header.

Tip: you can replay this later with Ctrl+Shift+T.
